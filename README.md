# WanderVista - Luxury Bhutan Travel 🏔️

A premium travel experience platform designed to showcase the beauty and serenity of Bhutan. Built with cutting-edge web technologies to provide a fluid, immersive, and high-performance user experience.

## ✨ Key Features

- **Immersive Hero Experience**: High-quality video backgrounds and smooth typography.
- **Micro-Animations**: Powered by Framer Motion for a premium feel.
- **Glassmorphism UI**: Modern aesthetic with blurred backgrounds and soft shadows.
- **Fluid Typography**: Responsive font scaling for all screen sizes.
- **Seamless Navigation**: Multi-step booking process and detailed destination views.
- **Vercel Optimized**: Configured for high-speed delivery and SPA routing.

## 🛠️ Tech Stack

- **Core**: React 19, TypeScript
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router 7
- **Build Tool**: Vite 7

## 🚀 Deployment

This project is optimized for **Vercel Deployment**.

1. Connect your repository to Vercel.
2. The `vercel.json` ensures all SPA routes are correctly handled.
3. Build command: `npm run build`
4. Output directory: `dist`

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
